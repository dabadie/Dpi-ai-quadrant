Methodology

  • Sources: DPI Map + UN EGDI snapshots, stored as local JSON in /data.
  • Scoring is deterministic and auditable.
  • Base DPI score = 70% DPI Map + 30% EGDI.
  • Base AI score is a conservative estimate: 0.6 × state capacity score.
  • Manual overrides always take precedence and are stored in localStorage only.
  • Missing data lowers confidence and completeness; values are never fabricated.
